# 1-3. PLG (Product-Led Growth)

## 핵심 정의
Product-Led Growth = 제품 자체가 획득, 전환, 확장, 유지의 주 엔진이 되는 전략
Sales-led(영업 주도), Marketing-led(마케팅 주도)와 대비되는 3번째 GTM 모델


### PLG 4대 축 (OpenView Partenrs 정의) :
1. Time to Value (TTV) : 가입 후 "아하 모먼트"까지 걸리는 시간 / median TTV
2. Self-serve : 영업 없이도 가치 경험, 결제 가능 / self-serve 비율
3. Viral/Network effects : 사용자가 늘수록 제품 가치 증가 / K-factors, NPS
4. Expansion : 사용량 기반, 팀 확장 기반으로 매출 증대 / NRR, Net dollar retention


### Freeminum vs Free Trial vs. Reverse Trial:
1. Freemium : 무표 티어 영구 (Notion, Figma) / 대량 funnel / 전환율 낮음
2. Free Trial : 14 ~ 30일 유료 체험 (Linear) / 명확한 압박 / trial abuse
3. Reverse Trial : 가입시 유료 기능 전면 개방 -> 일정 기간 후 freemium 다운그레이드 (최근 트렌드, Slack이 시초) -> 이 방향이 좋아보임. / activation 강력 / 가치 인지 어려움

- Activation (활성화) = PLG의 심장. 신규 가입자가 "아하 모먼트"를 경험해서 "진짜 유저"가 되는 전환점.\

"[X일 안에] [Y이라는 행동을] [Z번] 한 유저"

> 좋은 Activation 정의의 3대 조건:
> 1. 명확한 숫자로 표현 가능 (모호하면 실험 불가)
> 2. 유지율과 강한 상관
> 3. 온보딩에서 도달 가능한 합리적 난이도

예) - AI SaaS 
   : 대학 입시 AI 툴을 Saas로 만든다면?
   1. TTV 목표 : 가입 후 3분 내 첫 리포트 생성
   2. Acitvation 이벤트 : "첫 진단 리포트 받고 1개 대학 북마크"
   3. Expansion : 무료 3회 진단 -> 유료 무제한 + 학원 공유 기능 (팀 단위 확장)
   4. Reverse Trial : 가입 시 7일간 프리미엄 전 기능 -> 이후 Freemium 전환
